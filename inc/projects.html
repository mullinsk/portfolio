Computer Embroidery

Python based system that takes image input and uses openCV to analyze the image and Context Free Art to create a unique set of inspired designs. The new designs are analyzed and compared to the original image, and the most similar design is sent to be embroidered.

MedTracker: Mobile App Prototype

This UX protype was developed for the course Human Computer Interaction. The prototype went through the entire process of User Experience design, including requirements interviews, Paper prototyping, cognitive walkthroughs, and user testing. The final product is a front-end app with the capability to complete three use cases within the experience of adding prescription medicine to the app.
